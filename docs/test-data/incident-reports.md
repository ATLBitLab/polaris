# Fake incident report narratives

These are fictional first-person accounts derived from the records in `src/lib/fake-incident-reports.ts`. Each chunk below weaves together when, where, who, what happened, how the contact was made, what was demanded, the foreign-government connection, evidence on hand, and how urgent it felt — written the way a person might type it into the incident report form. Copy any block to seed a test submission.
